Question-1: What are the different ways to select an element in the DOM?

There are several ways to select elements in the Document Object Model (DOM). Some of the most common ones are:
* getElementById()
* getElementsByTagName()
* getElementsByClassName()
* querySelector()
* querySelectorAll()
* getElementsByName()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML lets us get or change the HTML inside an element, including tags.
innerText gives only the visible text, ignoring hidden parts and following styles.
textContent gets all the text, even hidden ones, but treats everything as plain text.

Question-3: What is event delegation in the DOM?

Event Delegation is basically a pattern to handle events efficiently. Instead of adding an event listener to each and every similar element, we can add an event listener to a parent element and call an event on a particular target using the .target property of the event object.

Question-5: How do you create, add, and remove elements using JavaScript?

* Create an Element: document.createElement()
* Add an Element: document.body.appendChild()
* Remove an Element: document.body.removeChild()
